CloudServers and services

The term Cloud stems from early network diagrams in which a cloud was used to represent a vast network, such as a wide-area network (WAN). The cloud soon came to be associated with the Internet as a whole, and the two concepts are now interchangeable.

Clouds are a term that refers to a collection of internet-connected servers that can be rented as part of a software or application service, web hosting, data hosting and sharing, software or program use are all examples of cloud services.

Cloud services are a broad category of internet-based services that are made available to businesses and customers on demand. These services are designed to provide easy, low-cost access to applications and resources without the need for internal infrastructure or hardware. Most employees use cloud services throughout the workday, whether they realise it or not, from checking email to collaborating on papers. Infrastructure, platforms, and software that are hosted by third-party providers and made available to consumers via the internet are known as cloud services, Users only need a computer, an operating system, and internet access to access cloud services.

What is a cloud server

A cloud server is a centralised server resource that is hosted and distributed across a network (often the Internet) and made available to many users on demand. Cloud computing infrastructure enables cloud servers to be located anywhere in the world and deliver services remotely.

Cloud servers are a crucial component of cloud computing. The widespread deployment of server virtualization has aided cloud computing's rise and continued expansion.

The creation of a cloud server is made possible via virtualization. A hypervisor is a piece of management software that connects and virtualizes physical servers by abstracting and pooling their resources to create virtual servers. These virtual resources can then be automated and made available via the cloud for internal or cross-company use.

Clouds have grown in popularity throughout the years, and I believe that everyone in this day and age has used or is utilising something related to clouds/cloud servers.



The following are some of the cloud's current features:

On demand self service
the most significant and valuable components of Cloud Computing since it allows the user to keep track of the server's uptime, capabilities, and network storage allocation. This function also allows the user to keep track of the computer's capabilities.
Simple Maintenance
The servers are simple to operate, and downtime is minimal, if not non-existent in some circumstances. Every time a new version of Cloud Computing is released, it improves exponentially.
Large network access
With the use of a device and an internet connection, the user can view the cloud's data or upload data to the cloud from anywhere. These capabilities are available throughout the network and can be accessed over the internet.
Security
One of the most appealing aspects of cloud computing is its security. It takes a snapshot of the data stored so that it doesn't get lost if one of the servers is broken. The data is saved on storage devices that cannot be hacked or accessed by unauthorised individuals. The storage service is efficient and trustworthy.
Resource Pooling
It signifies that the Cloud provider used a multi-tenant architecture to pull computing resources to provide services to numerous clients. Various physical and virtual resources are assigned and reassigned based on the needs of the consumer.
Automatic System
automatically analyses the data required and provides metering.
Economical
It is a one-time expenditure because the firm (host) must purchase storage and just a portion of it can be shared among several companies, saving the host money on a monthly or annual basis.
Edge Computing
Edge computing is a distributed information technology architecture in which client data is processed as close to the originating source as possible at the network's perimeter.
It's all about the location when it comes to edge computing. Data is produced at a client endpoint, such as a user's computer, in traditional enterprise computing. That data is transferred through a wide area network (WAN), such as the internet, to the corporate LAN, where it is stored and processed by an enterprise application. The work's results are subsequently sent back to the client's endpoint. For most common commercial applications, this is still a tried-and-true client-server computing architecture.
Micro Server
A micro server is a small server appliance that functions similarly to a traditional server. Micro servers are off-the-shelf computers that are designed to be simple to set up and maintain. The manufacturer will preinstall and configure the operating system, hardware, and applications on a mini server.
Availability
The Cloud's capabilities can be tailored to the needs of the user and greatly expanded. It evaluates storage utilisation and, if necessary, allows the user to purchase more Cloud storage for a modest fee.

CyberSecurity

Cybersecurity is the practise of securing systems, networks, and programs from digital threats is known as cybersecurity. These cyber-attacks are typically targeted at gaining access to, altering, or destroying sensitive data, extorting money from users, or disrupting normal corporate activities. Because there are more devices than humans nowadays, and attackers are growing more inventive, putting in place effective cybersecurity measures is very difficult.

There are a few different types of cybersecurity threats:

Phishing
Phishing is the act of sending fake emails that look like they came from a credible source. The main goal of phishing is to steal sensitive information such as credit card details and login credentials. It's the most known form of cybercrime.
Ransomware
Ransomware is malicious software that encrypts and holds files hostage. Its goal is to extort money by blocking access to data or the computer system until a ransom is paid, but there is no assurance that your files or system will be recovered. There are a fair few types of ransomwares but some of the well know ransomwares are
  • Ryuk
  • Maze
  • REvil (Sodinokibi)
  • Lockbit
  • Dearcry
Ransomware must get access to a target system, encrypt the files on it, and demand a ransom from the victim in order to be successful. While the specifics of implementation differ from one ransomware variation to the next, all three stages remain the same.
Malware
Malware is a form of software that aims to gain unauthorised access to a computer or do harm to it. Malware can infect personal computers, cell phones, tablets, servers, and even equipment, to name a few. The first sort of malware to be produced was a computer virus. Malware infects a computer by deceiving users into downloading and installing a program they should not. When the user clicks or installs the software, the malicious code does action the user did not expect or plan, such as self-replication in various portions of the file system. Installing software that collect keystrokes or commandeer system resources, frequently without the user's knowledge, while significantly slowing down the machine and posing a security risk. Access to files, programs, and even the operating system can be restricted.

Malware refers to various sorts of malicious software, including viruses, and cybercriminals employ it for a variety of reasons, including stealing information.
Such as:
  • Persuading a victim to provide personal information in order to commit identity theft
  • Stealing credit card or other financial information from the target
  • Taking control of a large number of computers in order to perform service dos attacks against other networks
Social Engineering
Adversaries employ social engineering to deceive you into divulging crucial information. They can demand a monetary payment or get access to your personal information, it can also make you more inclined to click on links, download malware, or believe a malicious link, social engineering can be used with any of the risks outlined above.

Everyone benefits from improved cyber defence strategies in today's linked society. A cybersecurity attack on an individual level can result in everything from identity theft to extortion attempts to the loss of crucial data such as family photos. Critical infrastructure, such as hospitals, and financial services organisations. It is critical to keep these, and other institutions secure in order for our society to function.

For example, instal anti-virus software, create a strong password, and use Fingerprint or Face ID whenever available. Raspberry Pi, Arduino, and other compact Singleboard Technologies

Compact single board computers and controllers have recently been growing in popularity and many new uses for them are being implemented and experimented with by professionals, hobbyist tinkerers and students. Most notable of the single board computer market is the Raspberry Pi and other SBC’s that have been inspired by them. For the controllers the most notable is the Arduino which is a compact circuit board with a micro-controller chip, the board itself is often referred to as a microcontroller.

Eben Upton the founder of Raspberry Pi Foundation noticed that there were less people enrolling in computer science as opposed to past years.

"The numbers were just awful, they had dropped off a cliff. We'd gone from having about 600 applicants for 80 or 90 places at the turn of the century down to under 250 people." Eben Upton stated in an interview for TechRepublic.
(Nick Heath 2018)

They wanted to make a fully functional computer that could be mass produced and distributed at an affordable price for a wide range of demographics so that every household might own an easy to program device capable of computing tasks and children could be exposed to computers and programming at a younger age (Raspberry Pi Foundation 2019).

The Ivrea Interaction Design Institute created the Arduino for students to assist them in learning to program and set up hardware (Arduino 2021). As they gained traction they began offering more board options and Shields or ‘daughter board’ which are expansion hardware that can add a wide selection of specific functions such as ethernet, smoke detection, camera, or a joystick to the main board. This customizability has popularized the Arduino with hobbyists and professionals alike.

What can be done now

Though the Raspberry Pi was created with the intention of being a mass-produced microcomputer for learning, it has since become popularised as a computer that powers many creative projects.

some of which include:

  • A system with an OS to function as a regular PC.
  • Hosting a website. With the appropriate web server software on a Pi you can store HTML files so that people can view your websites or you can use it as a home server and share storage between computers.
  • Creating retro gaming consoles or putting them inside of an arcade system and downloading emulators to play backed-up game files from your retro game cartridges or perhaps classic PC games.
  • Home media centres hooked up to a TV as the Pi can load TV shows or movies from a hard drive using software like Kodi or an alternative.

Arduinos could be used for controlling almost everything inside of your home perhaps commanded by a Raspberry Pi for automatic projects.

Some of which include:

  • Robots. Arduinos can be used to control robots and give them basic commands via a controller or to autonomously perform certain actions when a condition is met for example: a moving robot might backpedal once its movement is impeded and turn to avoid an obstacle.
  • A digital thermometer. With a thermometer expansion and display you can have the Arduino give you a temperature reading and can even log the temperatures.
  • A camera. A camera could be used in conjunction with a range of sensors to activate when a trigger happens such as temperature change, motion detected, switch activated or at a certain time of day.

Raspberry Pi shares it’s board layout as open source material on its website as well as offering Pi OS, an open source operating system based on Debian Linux. This adds to the customizability of the Raspberry Pi as you can modify the operating system or install a completely new OS if you desire.

Both of these organizations' devices are distributed by retailers or made by a third party as they are open source since both their design is openly available to anyone and their programming software is also available for free and is open source as well.

What is likely to be able to be done soon(say in the next 3 years?)

Due to the compact nature and their ability to control devices in complex ways, in the near future we will likely see similar SBCs or Microcontrollers in everyday use. They could be implemented commercially in the environment to control everything automatically for example: street lights could be disabled if no sensors have been triggered by a human to turn them on, or if the light level drops to a low level. Perhaps a buildings climate system could be controlled by thermometers situated around the building and disabled via cameras that keep track of where people are inside the building.

What technological or other developments make this possible
As technology is seeing great advancements and computers are becoming more capable and more hardware is being added to them, so too is the cost of acquiring them is increasing, Raspberry Pi, Arduinos, and their counterparts go against the grain and use technological advancements and scales them back into a lower powered, smaller form-factor and affordable package which takes advantage of modern technology in a unique way by making them almost disposable.

What is the likely impact
The ability to manufacture these compact SBCs and microcontrollers at a low price enables individuals or organisations to acquire them and modify them for niche purposes as opposed to developing their own complex circuit boards or contracting an engineer to create them which would be expensive to a degree whereas a home tinkerer or hobbyist would not even be able to access this technology and an organisation may decide it is not worth developing the board due to costs.

These basic skills will lay the groundwork for them advancing their education down an IT path in the future if they desire.
