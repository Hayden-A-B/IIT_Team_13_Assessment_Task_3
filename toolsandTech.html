ToDoTopia's concept is to use APIs to connect workplace productivity, work in progress, and time tracking apps to our video game. Jira's Kanban Board will be the first application we link to. We'll utilise JSON to link task progress, status, weight, and which people are responsible for completing them. If ToDoTopia was to be created the tools and software required would be.

  • Jira’s Kanban Board
  • JSON
  • Amazon Web Services (AWS)
  • GameLift
  • Unity
  • API’S (Postman or Swagger)
  • Visual Studio
  • Microsoft SQL server

Majority of the software required by Team Thirteen to create ToDoTopia all have free versions or offer trials periods and once they end you will have to upgrade to the professional versions available.

Amazon Web Services has over 100 products, and you can start building on AWS for free. Depending on what you require. AWS has three different free versions

  • Free trials - Short-term free trial offers begin when you activate a certain service (Limited)
  • 12 months free - These incentives are valid for 12 months after your original AWS sign-up date. (Covers all packages for 12 months then a fee is required)
  • Always free - These free tier offerings are accessible to all AWS users and do not expire. (Very limited)

Start with game-ready templates designed to help you customise and share your first game and take the fun of game creation to a whole new level are just a few of the things you can do with Unity Hub (free version). You may easily add your own distinctive style and ideas to your game by using Creative Mods, which are mini adventures. Unity Pro (premium version) is a lot more advanced than Unity Hub; it lets you to start from scratch and contains a lot more critical capabilities for game development, such as

  • Core Unity real-time development platform
  • Bolt Visual Scripting
  • High-end art asset pack
  • Three seats of Unity Teams Advanced
  • Create and deploy to closed platforms
  • Splash screen customization
  • Integrations with Collaboration Tools
  • Source code access +
  • Build Server license capacity +

Visual Studio is available in three different versions: Visual Studio Community, Visual Studio Professional, and Visual Studio Enterprise. Enterprise is the best tier of service because it includes more advanced testing tools, advanced debugging and diagnostics, better Collaboration Tools and Features, and advanced Cross-platform Development tools.

Jira’s Kanban board offers a free 30-day trial period until you will have to then go purchase their premium version to continue using their software.

GitHub is divided into three service tiers: GitHub Free, GitHub Team, and GitHub Enterprise. GitHub Free has everything you need to get started, including unlimited public and private repositories, automatic security, and version updates, 2,000 CI/CD minutes each month, and 500MB of package storage. GitHub Team adds GitHub Code spaces, protected branches, multiple reviewers in pull requests, draught pull requests, code owners, pages, and wikis, 3,000 CI/CD minutes per month, 2GB of package storage, and web-based support to the GitHub Free package. GitHub Enterprise is the most advanced version of the platform. It has all the features of GitHub Free and Enterprise, plus enterprise managed users, user provisioning through SCIM, enterprise account to centrally manage multiple organisations, environment protection rules and secrets, audit log API, SOC1, SOC2, type 2 reports annually, FedRAMP tailored authority to operate (ATO), SAML single sign-on, advanced auditing, GitHub connect, 50,000 CI/CD minutes/month and 50GB of packages storage.

Although Postman is a free API, there are premium versions available that can enhance your experience with the app. For up to three people, Postman (Free) offers all the core tooling as well as collaboration. Postman (Basic) comes with unlimited collaboration for teams of four or more people, 30 days of collection recovery, one custom domain, ten calls to the Postman API, and ten integrations. Postman (Professional) adds single sign-on (SAML), 90-day collection recovery, basic roles and permissions, private workspaces, static IP addresses for API testing, 5 custom domains, 100 calls to the Postman API, and 50 integrations to the basic functionality. Postman (Enterprise) is the most expensive package available and includes all of the features of the other packages as well as additional features such as identity and access management, including user governance, automated provisioning and deprovisioning via SCIM, reporting and analytics, deployment control, private and partner networks, 25 custom domains, 1000x Postman API calls, 100 integrations, including Enterprise Integrations with GitHub and GitLab, domain capture, and enterprise integrations with GitHub and GitLab.
